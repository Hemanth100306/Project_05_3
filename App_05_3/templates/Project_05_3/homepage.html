Real-time Computer Vision

SignTalk

Problem

Millions of hearing-impaired individuals in India use Indian Sign Language (ISL) for communication. The absence of real-time digital translators creates a significant accessibility gap.

Solution

Built a real-time ISL translator using MediaPipe's hand-landmark detection model and OpenCV for live webcam processing. The system classifies hand gestures into ISL characters with sub-100ms latency per frame, achieving over 90% accuracy on trained gestures.

Key Features
21-Landmark Hand Tracking ISL Gesture Recognition Live Webcam, ~90% Accuracy Real-time Processing
Python OpenCV MediaPipe TensorFlow
GitHub
Optimization & Operations Research

OptiLoader

Problem

Resource allocation problems in logistics and operations are NP-hard. Manually solving linear programming models is infeasible at scale and prone to suboptimal decisions.

Solution

Built an interactive LP optimization tool using SciPy's linear solver and Matplotlib for visual output. Users define constraints and objective functions via a simple interface; the solver returns the optimal resource allocation with a graphical feasible-region plot.

Key Features
LP Solver (SciPy) Feasible Region Plot Constraint Visualization Resource Allocation
Python SciPy NumPy Matplotlib
